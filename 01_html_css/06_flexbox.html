C - direction

Flexbox nous permet d'agencer ces éléments dans le sens que l'on veut flex-direction, on peut les positionner verticalement ou encore les inverser. Il peut prendre les valeurs suivantes :
-> row : organisés sur une ligne (par défaut)
-> column : organisés sur une colonne
-> row-reverse : organisés sur une ligne mais en ordre inversé
-> column-reverse : organisés sur une colonne, mais en ordre inversé
